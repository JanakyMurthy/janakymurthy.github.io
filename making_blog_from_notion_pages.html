Making a blog from Notion pages

Posted on Sun, Jul 25, 2021 how-to web blogs

I have been on an eternal search for an easy to create and deploy static site generator. I started with Pelican and hosted my blog on GitHub. Later I moved to Hugo which was faster, well maintained and has a less tedious workflow. The problem was I kept forgetting simple stuff - which directory am I supposed to save the images, what are the set of commands that needs to be executed to compile and push stuff. I can create a shell script for all of this - but again I am very clumsy - I end up deleting important scripts as well (huh, then what's git for? I should stop giving excuses). Had the workflow been simpler, looking up the documentation would have taken much less time. Another drawback was I used to write my blog post in some markdown editor which was not a pleasant experience.
